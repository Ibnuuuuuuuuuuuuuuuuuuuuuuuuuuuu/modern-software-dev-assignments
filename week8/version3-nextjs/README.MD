## Stack
- Backend: Node.js
- Framework: Express.js
- Database: In-memory (Array-based)

## Installation & Setup
1. Masuk ke direktori: `cd week8/version3-nextjs`
2. Install dependensi: `npm install`
3. Jalankan aplikasi: `node server.js`

## Features
- Basic CRUD functionality for notes.
- Lightweight and fast JavaScript backend.
